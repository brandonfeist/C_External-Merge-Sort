mining algorithm

1. College of Mathematics and Computer Science, Chongqing Three Gorges University, Chongqing, 404000, P.R. China

Abstract—this paper proposes an algorithm of spatial association 
rules mining based on orientation frequent, which is suitable for 
mining transverse spatial association. The algorithm uses the 
method of orientation frequent item set to generate frequent 
candidate items by down-up strategy, which is able to efficiently 
reduce repeating computing and redundant operation to improve 
the efficiency of algorithm. The algorithm is applied to Grid 
Visual Manage System based on GIS to analyze Grid 
Malfunction, which is able to efficiently improve the efficiency of 
system. 
Keywords- orientation frequent item sets; spatial association rules; 
binary; data mining; Grid analyzing
